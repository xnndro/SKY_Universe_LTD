# Step
- 1. install the dependencies
```sh
composer install
```
- 2. copy .env.example to .env
```sh
cp .env.example .env
```
- 3. generate key
```sh
php artisan key:generate
```
- 4. run the server
```sh
php artisan serve
```
- 5. open the browser
```sh
http://localhost:8000
```
- 6. register and login
- 7. enjoy the app
